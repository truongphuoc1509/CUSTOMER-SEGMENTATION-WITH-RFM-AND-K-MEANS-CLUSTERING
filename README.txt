Final project about Customer Segmentation with RFM and K-means Clustering

Final project use RFM and K-means Cluster methods to devide customer segment and compare their performance. Project use dataset transaction of Momo, collected from January 2021 to March 2022

There are five files besides this README:
1. DataTransaction_Jan2021_to_Mar2022.xlsx - Our dataset
2. Final.doc 
3. Final.pdf
4  Final.ppt
5  Final.ipynb - File code

Prerequisites

Before you continue, ensure you meet the following requirements:

* You have installed the 3.9.7 version of Python, IDE or Anaconda/ Google colab
* You have installed two packages: yellowbrick.cluster and squarify

Type the following code in cell and run to install them:
%pip install yellowbrick
%pip install squarify
